11. Profiling

a) Compilez le jeu avec la configuration "Profile", puis lancez le jeu.

b) Prenez une capture PIX. ( https://devblogs.microsoft.com/pix/download/ )

  • Dans File > Settings > Symbol / PDB Options > PDB Search Paths, ajoutez srv*c:\symcache*https://msdl.microsoft.com/download/symbols (même s'il y a un texte d'erreur).
  • Dans Local Machine > Capture > Attach (à gauche), cherchez le nom de l'exécutable (Dino_PrenomNom.exe), et cliquez dessus.
  • Décochez la case For GPU Capture, puis appuyez sur le bouton Attach.
  • Dans Capture, cochez CPU Samples avec 8k / sec, puis appuyez sur Start Capture.
  • Jouez pendant quelques secondes.
  • Cliquez sur Stop Capture.
  • Allez dans l'onglet en haut Timing 1.wpix, puis dans l'onglet Timeline

c) En haut à gauche de la fenêtre, vous pouvez voir Reload Symbols. Cliquez dessus. Cette étape télécharge la première fois environ 250 MiB de données dans le dossier C:\symcache. Dépendamment de la connexion, cela peut prendre plusieurs minutes.
